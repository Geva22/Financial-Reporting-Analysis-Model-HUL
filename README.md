# Financial Reporting & Analysis Model – HUL

## 📌 Overview
This project demonstrates a financial reporting and analysis model built in Excel using the financial data of Hindustan Unilever Limited (HUL). The objective is to replicate real-world financial reporting practices and analyze company performance.

---

## 📊 Key Features

- Preparation of Financial Statements:
  - Income Statement (Profit & Loss)
  - Balance Sheet
  - Cash Flow Statement

- Financial Analysis:
  - Year-on-Year Growth Analysis
  - Revenue and Expense Trends
  - Profitability Insights

- Excel Skills Used:
  - Formulas and Functions
  - Data Structuring
  - Basic Financial Modeling

---

## 🧠 Key Learnings

- Understanding of financial reporting structure
- Practical application of accounting concepts
- Ability to analyze financial performance
- Improved Excel and data handling skills

---

## 📁 File Included

- HUL Financial Reporting Model (Excel)

## 🚀 How to Use

1. Download the Excel file

   - Income Statement
   - Balance Sheet
   - Cash Flow
   - Analysis
